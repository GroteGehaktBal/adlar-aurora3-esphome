# Support

This is an unofficial community project maintained on a best-effort basis.

## For Questions

Open a GitHub issue and include:

- Heat pump model.
- JÅN module details if known.
- ESPHome version.
- ESP board and RS485 module.
- Wiring summary.
- Relevant logs with secrets removed.

## For Installation Or Safety-Critical Problems

Contact your installer, electrician, heating engineer or the heat pump vendor. This project is not a replacement for professional installation, service or safety checks.
